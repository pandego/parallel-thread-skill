# Purpose

Command template for Gemini CLI agent processes in pthd.

## Variables

COMMAND_TEMPLATE: gemini -y -i '{prompt}'

## Instructions

- Before first use, run `gemini --help` to understand options
- `-i` flag enables interactive mode (must be right before prompt)
- `-y` (yolo mode) enables autonomous operation (no approval prompts)
- Aliases: gem, gems, gemini, gemini-cli
