# Purpose

Command template for Codex CLI agent processes in pthd.

## Variables

COMMAND_TEMPLATE: codex --dangerously-bypass-approvals-and-sandbox '{prompt}'

## Instructions

- Before first use, run `codex --help` to understand options
- Interactive mode (no `-p` flag) - prompt as positional argument
- `--dangerously-bypass-approvals-and-sandbox` enables autonomous operation
- Aliases: codex, codex-cli
